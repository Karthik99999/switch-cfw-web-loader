How to use SHAIO

Honestly, the program practically teaches itself, but I'll give instructions anyway.

1. Get the latest release of SHAIO here

2. Install it if you got the exe, or unzip the file if you got the portable zipped file.

3. Open SHAIO, then click "Payload Tool", and press ok on the popup (first use only).

4. Select "Install Drivers" in the bottom left corner. Allow the app to make changes (needs admin access), then complete the wizard.

5. Put your switch into RCM mode and plug it into your computer.

6. Click the text box at the top and select a paylod, or upload one with the "+" button, then click "Launch", and it should be done!

The Web Launcher will work on Linux, however, I suggest you do this first so we don't have errors later on:

Make a file at /etc/udev/rules.d/50-switch.rules With the following contents:
SUBSYSTEM=="usb", ATTR{idVendor}=="0955", MODE="0664", GROUP="plugdev"

These Operating Systems will work as is. Move on to the hacking!
