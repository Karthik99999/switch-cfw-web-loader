The Web Launcher will NOT work on Windows. Instead, I recommend you use TegraRcmSmash.

Official Instructions can be found here, but here is a basic setup:

Programs Needed

Go here and get the latest version of Zadig

Then go here and get the latest verison of TegraRcmSmash

Setup

1. Put your switch into RCM Mode

2. Connect your switch to your PC, then open Zadig.

3. Select "APX" in Zadig, then select libusbK (v3.0.7.0) as your driver type in the text box.

4. Select "Driver Install". Once it's done, check the device manager and see if "APX" is under libusbK USB tree.

Using your payload

After the setup, it's simple. Unzip the file you downloaded for TegraRcmSmash. Then open the x64 folder (or Win32 if you have a 32 bit OS). Then drag and drop your payload on the .exe (Make sure your switch is connected of course, which is the same method as seeing if the driver was installed) and it should work!

The Web Launcher will work on Linux, however, I suggest you do this first so we don't have errors later on:

Make a file at /etc/udev/rules.d/50-switch.rules With the following contents:
SUBSYSTEM==\"usb\", ATTR{idVendor}==\"0955\", MODE=\"0664\", GROUP=\"plugdev\"

These Operating Systems will work with no modifications. Move on to the hacking!
